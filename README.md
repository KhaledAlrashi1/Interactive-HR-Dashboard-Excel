# Interactive HR Dashboard (Excel)

An Excel-based interactive HR analytics dashboard with PivotTables/Charts, Slicers, and Power Query.

**Live demo:** [Open in your browser](<YOUR_OFFICE_WEB_VIEWER_OR_PAGES_LINK>)  
**Download workbook:** `excel/Interactive-HR-Dashboard.xlsx`

## Highlights
- KPIs: Average Salary, Average Age
- Salary band analysis (T1–T5) with bottom/average/top comparisons
- Filters: Manager, Department, Region, Gender (via Slicers)
- Distribution of employees by age
- Clean data pipeline (Power Query) + protected views

## Data
- Sample (anonymized) employee dataset with columns like:
  `EmployeeID, Department, Region, Gender, Age, FTE_Salary, Manager, SalaryBand`
- No PII; values are synthetic/anonymized for demo.

## How to use
1. Open in Excel or Excel for the Web.
2. Use slicers on the right to filter views.
3. Refresh (Data → Refresh All) if you connect your own CSV.

## Notes
- Built in Microsoft Excel (PivotTables/Charts, Slicers, Power Query).
- Tested in Excel for the Web (interactivity supported).
